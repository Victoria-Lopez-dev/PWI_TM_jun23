Definicion DOM

DOM(Document Object Model) es la interfaz entre javascript y HTML+CSS.

Es una representación completamente orientada al objeto de la página web que nos proporciona otras formas de presentar, guardar y manipular este mismo documento

Los objetos del DOM modelizan tanto la ventana del navegador como el historial, el documento o página web, y todos los elementos que pueda tener dentro la propia página, como párrafos, divisiones, tablas, formularios y sus campos, etc
